구조 (단일 상태)

진입=대기(idle, loop) → 말 안 할 땐 대기 루프, 말할 땐 그 위에 Ditto 얼굴. 말하기(speak, loop+can_talk) = 헤드토킹 베이스. 동작(gesture, 비loop) = LLM이 트리거하는 표정/제스처.

종류는 loop / can_talk 조합으로 자동 판별. 포즈/전환/상태(앉기·눕기 등)는 다음 버전.

사용

Naia 로드(기본) / .nva 열기 → 좌측 animations 편집(클립 교체·loop·can_talk·face_bbox) → 우측 플레이어에서 미리보기 → cascade 연결로 실시간 발화 확인 → 내보내기.

배경 · 알파

알파(VP9 yuva420p webm) 클립은 배경 위에 그대로 합성. 미리보기 배경색으로 확인. cascade enable_alpha 면 발화도 투명.

LLM 트리거 (VRM 유사)

동작 애니에 intent/triggers + 최상위 expressions 맵(감정→animation). naia-shell LLM이 대화 감정으로 애니 선택.
